# Hero Trainer

To install and run this app :

1- Take the apk in APK/ and install it on your phone.

2- If your phone/emulator is connected to android studio :

    a- flutter pub get
    b- flutter build apk --split-per-abi
    c- flutter install

    in the console.

Our application does only require internet connection for fight musics but it is an optional feature.
